# Library Management System

## Project Description
This is a Database Management System (DBMS) project for managing a library system.

## Features
- Manage Members
- Manage Books
- Manage Authors and Categories
- Issue and Return Books (Loans)
- Automatic Audit Logs using Trigger

## Database Tables
- Member
- Book
- Author
- Category
- Loan
- Librarian
- LoanAudit

## SQL Files
- tables.sql → Database structure
- insert.sql → Sample data
- queries.sql → SQL queries
- views.sql → Views
- triggers.sql → Triggers
